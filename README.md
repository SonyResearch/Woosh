# SFXFM

Public release of the Sound Effect Foundation model by Sony AI.

## Installation

Use [uv](https://docs.astral.sh/uv) to install SFXFM.

```bash
uv sync
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
